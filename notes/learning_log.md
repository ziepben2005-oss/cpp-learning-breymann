# Important Stuff
## Terminal commands
1. to change directory you type cd and then the directory e.g. cd ".\Folder1"
2. to compile and create program type: g++ -o test1.exe ".\test1.cpp"
   1. condition: must be in correct directory
   2. to define standart check e.g. c++23 type: g++ -std=c++23 -o test1.exe ".\test1.cpp"
## Cheatsheet
- '\n' : go to next line
- cin >> input1 >> input2; : 'cin' means input e.g. number input from keyboard, '>>' the arrow points to the object that should get an imput throu the keyboard
- cout << "text" << output1 : 'cout' means console output, the arrows are only for order of to be diplayed content
